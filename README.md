# Snake Game
It's a simple console based game implemented in CPP.

## Installation
In case you don't have git<br/>

`sudo apt-get install git`<br/>

Clone the repo<br/>

`git clone https://github.com/beyourwild7/SnakeGame.git`<br/>

Open the directory<br/>

`cd Tic-Tac-Toe`<br/>

## Usage
### Compilation
`
g++ Game.cpp
`
### Run
`
./a.out
`
